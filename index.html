Featured Projects

PCALib — predicting PCA recovery in neural population recordings

A Python library for neural population analysis. It enables you to assess and predict how accurately PCA will recover latent low-dimensional dynamics from neural recordings, given your number of neurons, number of trials, noise levels, and other experimental parameters. It also allows you to extrapolate to larger dataset sizes and to model how PCA accuracy scales.

  • Random matrix / spiked covariance theory + validation with large-scale simulations
  • Pipeline: fit → extrapolate → predict accuracy; supports realistic noise + heterogeneity
  • Python + JAX; designed for experimental planning
